# Student Success Prediction using Machine Learning

## Project Overview

This project uses Logistic Regression to predict whether a student will pass or fail based on study habits and academic performance.

## Features

* Data preprocessing using Label Encoding and Standard Scaling
* Logistic Regression model training
* Model evaluation using Classification Report and Confusion Matrix
* User input prediction for new students

## Dataset Features

* StudyHours
* Attendance
* PastScore
* SleepHours
* Internet
* Passed (Target Variable)

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Seaborn

## How to Run

1. Install required libraries.
2. Place `student_success_dataset.csv` in the project folder.
3. Run the Python script.
4. Enter student details when prompted to get a prediction.

## Output

* Pass/Fail Prediction
* Probability of Passing
* Confusion Matrix Visualization
